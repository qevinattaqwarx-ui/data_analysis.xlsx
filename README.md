Excel Data Analysis Project – Qevin Attaqwa
Project Overview

This project is a data analysis case study using Microsoft Excel. The goal is to clean raw data and perform exploratory analysis using pivot tables to extract meaningful business insights.

The project simulates real-world data analysis workflow: data cleaning, transformation, analysis, and reporting.

Tools Used
Microsoft Excel
Pivot Tables
Charts (Basic Data Visualization)
Project Structure

This Excel file contains multiple sheets:

Raw_Data: Original dataset before processing
Cleaned_Data: Data after cleaning and formatting
Pivot_Sales_By_Category: Sales analysis by category
Pivot_Monthly_Trend: Trend analysis over time
Top_Products_Customers: Top performing items or customers
Charts: Visual representation of key insights
Data Cleaning Process
Removed duplicate records
Handled missing or null values
Standardized data formats (dates, categories, text)
Ensured data consistency for analysis
Analysis Performed
Pivot table aggregation for category performance
Time-based trend analysis
Ranking of top and bottom performers
Segmentation of data for deeper insights
Data visualization using charts
Key Insights
A small number of categories contribute most of the overall results
Clear trends can be observed over time periods
Performance varies significantly across different segments
Certain categories consistently outperform others
What I Learned
How to clean and structure raw data in Excel
How to use pivot tables for data analysis
How to extract insights without programming
How to present data in a clear and structured way
How to Use This Project
Download the Excel file
Open in Microsoft Excel
Navigate through each sheet step by step
Explore pivot tables and charts
Closing Note

This project demonstrates foundational skills in data analysis using Excel, focusing on data cleaning, pivot tables, and business insight generation.
